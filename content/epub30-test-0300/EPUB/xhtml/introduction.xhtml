Introducción

Estado de este Documento

Actualmente, esta publicación es considerada [estable] por el Consorcio DAISY y se lleva a cabo en apoyo al trabajo del W3C EPUB 3 Community Group.

Esta publicación es una de las que conforman la Batería de Test de Accesibilidad EPUB 3.

Sobre este documento

Este documento contiene test que son fundamentales para la accesibilidad de los sistemas de lectura para usuarios con discapacidades. Los test avanzados están contenidos en publicaciones separadas que se encuentran en la batería de test.

Un sistema de lectura con accesibilidad básica debería posibilitar la lectura con los ojos, con los oídos o con los dedos. Esto significa que debería ser posible para los usuarios leer el contenido en las siguientes modalidades:

  • adaptando la pantalla, por ejemplo cambiando el tamaño de la fuente o la combinación de colores
  • leyendo el texto con lectores de pantalla o lectura en voz alta
  • leyendo el texto con una pantalla de braille efímero

Las etiquetas de los test incluyen una ID, por ejemplo, file-010. Estas ID de los test corresponden a las entradas de los formularios de evaluación de accesibilidad de www.epubtest.org

Cómo se realizan los test

Los sistemas de lectura serán evaluados por varios evaluadores usando diferentes combinaciones de tecnologías de apoyo. Habrá un informe sobre los test para cada combinación. Los evaluadores podrán realizar los test proporcionados en este libro mediante un teclado, ratón o gestos táctiles para la entrada de datos. Para la salida, los evaluadores podrán usar audio (mediante lector de pantalla y/o lectura en voz alta), adaptaciones visuales o braille efímero.

Si eres una persona que lleva a cabo lectura no visual con lector de pantalla o pruebas el sistema de lectura sin una pantalla, lo más probable es que realices los test de las siguientes secciones:

  • Test de funciones básicas
  • Test de navegación
  • Destacados, notas y marcadores
  • Test de lectura no visual

Si eres un lector vidente, si tienes baja visión o una diferencia de aprendizaje como la dyslexia, lo más probable es que realices los test de las siguientes secciones:

  • Test de funciones básicas
  • Test de navegación
  • Destacados, notas y marcadores
  • Test de adaptación visual
  • Test de lectura en voz alta

Cada test incluye una descripción de su objetivo, seguida de la proposición del test propiamente dicho, que puede evaluarse como verdadera o falsa.

El resultado de cada test debe registrarse en el campo correspondiente del formulario del sitio web. Los comentarios son de gran ayuda, y deben ser consignados en el espacio proporcionado en el formulario en línea.

Los test proporcionados en este libro deberían explicarse por sí mismos; no obstante, los evaluadores y desarrolladores podrían encontrar útil la consulta del documento de orientación adicional, que puede descargarse desde www.epubtest.org/instructions
