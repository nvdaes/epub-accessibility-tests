Documento test de conformidad de ePUB 3 accesible: Test fundamentales 0300

Estado de este Documento

Esta publicación actualmente está considerada [estable] por el IDPF y el DAISY Consortium.

Esta publicación es una de varias que conformarán EPUB 3 Accessibility Test Suite para contenido redimensionable.

Acerca de este Documento

Este documento contiene test que son fundamentales para la accesibilidad de los sistemas de lectura para usuarios con discapacidades. Los test avanzados están contenidos en un documento separado.

El presupuesto básico de la evaluación de accesibilidad es que todos los sistemas de lectura deberían admitir la lectura mediante los ojos, los oídos o los dedos. Debería ser posible para los usuarios leer el contenido de este documento mediante:

  • ajustes de la visualización, tales como el ajuste del tamaño de fuente y combinación de colores,
  • leer el texto con lectores de pantalla o TTS y
  • leer el texto en una pantalla de braille efímero.

Los sistemas de lectura deberían ser evaluados usando diferentes combinaciones de tecnologías de apoyo, y todos los test proporcionados en este libro deberían ser realizados. Se espera que un sistema de lectura accesible superará todos los test proporcionados en este libro, independientemente de los medios de entrada y salida usados en las pruebas. Quienes realicen las pruebas proporcionadas en este libro podrían llevarlas a cabo usando el teclado, ratón o gestos táctiles como entrada, y lector de pantalla/síntesis de voz propia, adaptaciones visuales o Braille como salida.

Convenciones

Las siguientes convenciones se usan a lo largo del documento:

Todos los test en este documento van precedidos de la etiqueta [REQUIRED], a fin de indicar que el test es fundamental para determinar la accesibilidad del Sistema de Lectura.

Las etiquetas de los test van seguidas de ID, tales como file--010. Estos IDs de los test corresponden a las entradas en el formulario en www.epubtest.org para la evaluación de accesibilidad.

Realizar los test

Cada test incluye una descripción de su propósito, seguida de la instrucción del test en sí, que puede evaluarse como verdadera o falsa.

El resultado de cada test de este libro debería registrarse en el campo correspondiente del formulario del sitio web, y si hay algún comentario, debería señalarse en el espacio proporcionado en el formulario en línea.
